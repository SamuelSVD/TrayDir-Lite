Program Settings

There are different settings that can be changed to alter the view or function of TrayDir

Windows Settings

  • Minimize On Close
    • Minimize screen when closing the main window through the X at the top of the form.
  • Hide On Minimize
    • Hide main form when minimized. It can be opened from the system tray by double clicking an icon in the system tray.
  • Start Minimized
    • Start the program minimized.
  • Start With Windows
    • Start TrayDir when loggin in to windows
  • Check For Updates on Startup
    • Check for updates online when starting TrayDir

Application Settings

  • Show Icons In Tray
    • The system tray menu items will display the icon of the item they are a shortcut to.
  • Menu Sorting
    • Folders Top: Always place subdirectories at the top of the menu
    • Folders Bottom: Always place subdirectories at the bottom of the menu
    • None: Do not differentiate between folders and files when creating the menus
  • Virtual Folder Menu Icon: Ability to select the between the following:
    • Yellow Folder icon
    • Blue Folder icon
