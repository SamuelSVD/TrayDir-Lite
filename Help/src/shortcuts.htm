Shortcuts

TrayDir has a few shortcuts available to allow you co customize the menu and items easier.

ShortcutDescription
CTRL+NCreate new File item
CTRL+NCreate new File item
CTRL+CCopy selected item
CTRL+VPaste selected item
CTRL+XCut selected item
CTRL+UPMove item up
CTRL+DOWNMove item down
CTRL+LEFTMove item out of Virtual Folder
CTRL+RIGHTMove item into Virtual Folder
