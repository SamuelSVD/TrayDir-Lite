Instance Options

Icon

The instance's icon can be changed. Doing so will change the icon that is used in the system tray. To do so, click on the Browse button to browse to any file whose icon you want TrayDir to use for this instance. All files are valid. I recommend downloading a .ico file from online and using that. It can help easily differentiate between instances in the system tray.

Options

  • Run As Administrator
    • Always attempt to run items in this instance as administrator
  • Show File Extensions
    • Show file extensions in the system tray menu items
  • Open Folders in File Explorer When Clicked
    • Open File Explorer in the folder path specified when a folder item is clicked from the system tray menu
  • Expand First Path
    • When only one item exists in the instance, and that item points to a folder, then the folder's contents will be used in the system tray menu instead.
  • Hide From Tray
    • Hide this instance's icon from the system tray
